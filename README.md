# JsHook - NeuralCloudHack

原本是使用 JsHook 和 frida-il2cpp-bridge 搞得 云图计划 的 Hack，参考了 FlxSNX 大佬曾经的代码，一直以来 **仅自用**。

但是自 3.0.0 起，游戏采用了 HybridCLR，该方案大概已 **完全失效**，故此开源并归档，留给后来人参考。
